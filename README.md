# Windows32-Utility

It uses Windows32 api's to support and provide functionality to query required details from the system and execute
provided options on the system.



Available Options - 
1. wmi          : Provides system information.
2. telnet       : Provides functionality to test connectivity with IP and Port.
3. ping         : Provides functionality to test connectivity with host.
4. service      : Provides managing window services through Service Control Manager (SCM).
5. netact       : Provides network activity details of the system with Process information.
6. usrsesn      : Provides functionality to get system current user session information and start process with specific user session.
